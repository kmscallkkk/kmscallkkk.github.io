Smart Factory Control System

Embedded Linux • Device Driver • MQTT • TLS • Optical Flow • OpenCV • Qt • Real-Time Control

Developed a Raspberry Pi–based Smart Factory Control System. Integrated Linux kernel drivers for servo and stepper motor control, MQTT-TLS communication, Optical Flow–based image analysis, and a Qt dashboard UI to achieve real-time equipment control and monitoring.

Infant Monitoring System

Raspberry Pi · Multithreading · TCP Communication · Sensor Fusion · Real-time Monitoring

Developed a real-time infant monitoring system that tracks temperature, heart rate, and environmental conditions. Implemented a multithreaded architecture with TCP communication for parallel sensor processing and instant alert transmission. Integrated a deep learning–based cry detection module that triggers motor vibration and white noise to calm the infant.

GPS Satellite Navigation Algorithm Portfolio

Python · Kalman Filter · Least Squares · Satellite Positioning · Visualization

Developed GPS/GNSS core algorithms from scratch, implementing both Single Point Positioning (SPP) and Kalman Filter–based tracking. Designed modular satellite positioning and DOP analysis to enhance precision under realistic orbital conditions. Validated through simulations, achieving up to 50% improvement in accuracy compared to standalone SPP results.
